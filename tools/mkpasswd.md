## Description

# Description

mkpasswd generates hashed passwords from plain text, using popular formats found in /etc/passwd and /etc/shadow files.

Available methods (-m <ALGO>):

- sha512crypt - SHA-512

- sha256crypt - SHA-256

- md5crypt - MD5

- descrypt -  standard 56 bit DES-based crypt(3)
