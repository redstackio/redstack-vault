# Description

rpcclient is a tool that can interact with SMB/CIFS MSRPC servers via an interactive shell. Once connected to a RPC server, rpcclient includes a number of useful commands including:

- queryuser - query a user's info

- querygroup - query a group's info

- enumdomusers - enumerate domain users

- enumdomgroups - enumerate domain groups

- samlookupnames - look up names

- lookupdomain - look up domain

- samlookuprids - look up usrs by RID
