# samdump2

## Overview

samdump2 is used to dump passwords hashes from Windows 2K/NT/XP/Vista SAM hives. It requires both the SAM and SYSTEM hives, as the SYSTEM hive contains the bootkey needed to decrypt the SAM hive.
