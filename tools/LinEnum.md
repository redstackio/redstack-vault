## Description

# Description

LinEnum is a Linux/BSD automated enumeration script written in Bash, which scans a local system for potential vulnerabilities, highlighting issues that warrant further investigation. From a high level, LinEnum's scans include:

- Kernel Information

- System Information

- User Information

- Privileged Access

- Environmental Variables

- Jobs

- Version Information

- Default/Weak Credentials

- Notable Files and Folders
