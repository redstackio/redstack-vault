# theHarvester

## Overview

Theharvester is mainly an OSINT enumeration tool, used to gather information on an organization, including e-mails, subdomains, hosts, employee names, open ports, and banners. TheHarvester can gather data from many OSINT sources, including Google, Bing, Baidu, LinkedIn, Twitter, Netcraft, and more,

# Description

Theharvester is mainly an OSINT enumeration tool, used to gather information on an organization, including e-mails, subdomains, hosts, employee names, open ports, and banners.
