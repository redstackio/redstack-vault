# Description

Covenant is a .NET command and control framework which aims to highlight .NET as an attack surface, and function as a collaborative command and control platform for red teams. Covenant is written in ASP.NET Core, and is compatible with MacOS, Linux, and Windows. Covenant also supports Docker, and is oftent he preferred way to run it.

Like other C2, Covenant includes many enumeration and exploit modules including:

- BypassAmsi

- BypassUAC

- Kerberoast

- Keylogger

- Seatbelt

- Screenshot

- SharpUp
