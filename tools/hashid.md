# Description

hashID is a tool written in Python 3.x which supports the identification of over 210 unique hash types using regular expressions. It is able to identify a single hash, parse a file or read files in a directory and identify the hashes within them.  hashID is also capable of including the corresponding hashcat mode and/or JohnTheRipper format in its output.
