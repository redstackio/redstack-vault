## ▓▓ ABOUT ▓▓

**Redstack Vault** is a structured offensive security knowledge base containing
**14,000+ records** of real-world penetration testing procedures, attack techniques,
executable commands, and pre-built attack chains—all mapped to the **MITRE ATT&CK**
framework.

Built for **red teamers**, **penetration testers**, and **security researchers** who
need a programmatic, queryable, and AI-ready dataset of offensive techniques.
