# Modify Windows Registry Keys

**Command** ([[Add a registry key / value]]):

```bash
reg add \\\ /v """ /t /d #Delete a key/value

```

**Command** ([[Delete a registry key]]):

```bash
reg delete \\\ /v ""

```
