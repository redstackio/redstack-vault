# Description

SSL/TLS can be enumerated for useful information, including certificate owner information (names, e-mails, etc), associated domain names, valid dates, etc. It is also possible to analyze the protocol to identify weak keys and cipher suites, downgrade opportunities, misconfigurations, and more.
