# rpcclient

**Command** ([[get information via rpc with username]]):

```bash
rpcclient -U username target-ip

```

**Command** ([[get information via rpc without username]]):

```bash
rpcclient -U "" target-ip

```

**Command** ([[sub-commands once connected]]):

```bash
srvinfo
lookupnames username

```
