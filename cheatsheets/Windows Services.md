# Windows Services

**Command** ([[Query service]]):

```bash
sc query

```

**Command** ([[Create service]]):

```bash
sc create <\\Target(optional)> binPath= type= share start= auto DisplayName=

```

**Command** ([[Delete service]]):

```bash
sc delete

```
