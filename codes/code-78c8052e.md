# Code Snippet 78c8052e

**Language**: ps1

```ps1
snaffler.exe -s - snaffler.log

The Snaffler command is used to find sensitive information on a network. The command has various arguments to specify the search criteria. The -s argument specifies to search for sensitive information. The -i argument can be used to specify a specific directory to search. The -d argument can be used to specify the domain to search. The -c argument can be used to specify the domain controller to use for the search. The -n argument can be used to specify specific computers to search.

For example:
# Snaffle all the computers in the domain
./Snaffler.exe -d domain.local -c <DC> -s

# Snaffle specific computers
./Snaffler.exe -n computer1,computer2 -s
```
