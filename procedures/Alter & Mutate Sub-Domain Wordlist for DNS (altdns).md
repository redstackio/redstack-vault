# Description

Alter and Mutate wordlists for potential subdomains by providing two lists as input to altdns.

This tool generates potential subdomains that can be resolved.

[altdns](https://github.com/infosec-au/altdns)





**Command** ([[altdns mutate sub domain wordlist and output resolved hosts]]):

```bash
altdns -i subdomains.txt -o data_output -w words.txt -r -s results_output.txt

```
