# Description

You can visualize the results of a scan using the viz sub-command. Our favorite visualize tool is D3, it is one of the best ways to programmatically map graphing database results to a website.

This command will create an amass_d3.html file inside the output directory.







**Command** ([[amass visualize assets using d3 from a previous scan]]):

```bash
amass viz -d3 -dir $_OUTPUT_DIRECTORY
```
