# Description

## Objectives

By using tools such as BloodHound, an attacker can gain a better understanding of the target environment and identify potential vulnerabilities and weaknesses. This information can be used to plan and execute more effective attacks, and to improve defenses against future attacks.



1. Analyze the collected data from the sharphound tool

2. Identify potential vulnerabilities or weaknesses

3. Plan an effective attack of the environment





# Instructions

1. Import data generated by a BloodHound ingestor using "Import Data" at the top right of the window

2.  Expand the top left menu and select "Queries"



![6ddf9299-4eab-40f1-bc80-21f85c7ea173.png](_assets/images/Mark/6ddf9299-4eab-40f1-bc80-21f85c7ea173.png)



3. Use the "Pre-Built Analytics Queries" to graph relationships. Look for paths to high value users, right click on the line indicating a relationship, and select "Help" to get more information.



![369ab609-57f2-4606-a39d-d9e98da25006.png](_assets/images/Mark/369ab609-57f2-4606-a39d-d9e98da25006.png)

BloodHound will identify and suggest attacks on the following ACE: ForceChangePassword, AddMembers, GenericAll, GenericWrite, WriteOwner, WriteDACL, AllExtendRgihts



4. BloodHound includes extensive documentation explaining  relationships, including "Abuse Info" which describes suggested attacks using popular tools such as PowerView and Mimikatz
